# Mental Health Detection Using Machine Learning
## Project Title
Detection of Depression and Anxiety Using Social Media Text Data with Machine Learning
## Description
This project aims to build a machine learning–based system to detect signs of depression and anxiety from social media posts. It uses Natural Language Processing (NLP) techniques for text preprocessing and feature extraction. Multiple machine learning and deep learning models are trained and evaluated to identify the best-performing model.
## Technologies Used
- Python  
- Pandas, NumPy  
- NLTK, SpaCy  
- Scikit-learn  
- TensorFlow / PyTorch  
- Streamlit  
## Project Structure (Planned)
- data/               → Dataset files  
- notebooks/          → Jupyter notebooks  
- src/                → Python scripts  
- models/             → Saved trained models  
- app.py              → Streamlit app  
- requirements.txt    → Required libraries
